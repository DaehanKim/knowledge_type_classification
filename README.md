# knowledge_type_classification

knowledge type classification using GRU+Attention mechanism

## How to run

 - put ```txt_clf_data.tsv``` and ```kor2vecxxx.checkpoint.ep0``` in the same directory as ```main.py```
 - run `python main.py`
